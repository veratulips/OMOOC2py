##1W-作业说明

###成品：请直接运行[```diary.py```](1wex1/diary,py)

####**三个功能模块：**

* 目录(menu)

    	欢迎来到你的日志
    	基本功能如下："
    	读取（r）
    	写入（w）
    	退出（q）
    	请选择您想做的事情(输入大写字母即可): 

* 读取日记(read)

    	程序将读取您以往的日记
    	blablabla
    	blabla
    	bla

* 写入日记(write)

    	您可以开始写日记了
    	回车可换行输入。退出请输入：q
        >
        
####总运行部分：

首先打印以往日记，然后出现目录，用户可选择想要做的事情，然后进入具体模块。
